QDomComment¶

The QDomComment class represents an XML comment. More…

Detailed Description¶

A comment in the parsed XML such as this:

<!-- this is a comment -->

is represented by QDomComment objects in the parsed Dom tree.

For further information about the Document Object Model see Level 1 and Level 2 Core . For a more general introduction of the DOM implementation see the QDomDocument documentation.

class QDomComment¶

QDomComment(x)

param x

QDomComment

Constructs an empty comment. To construct a comment with content, use the createComment() function.
