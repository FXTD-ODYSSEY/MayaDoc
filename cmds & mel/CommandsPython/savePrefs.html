command (Python) MEL version

savePrefs

In categories: Windows, Misc. UI

Go to: Synopsis. Return value. Flags. Python examples.

Synopsis

savePrefs([colors=boolean], [general=boolean], [hotkeys=boolean], [menuSets=boolean], [plugins=boolean], [uiLayout=boolean])

Note: Strings representing object names and arguments must be separated by commas. This is not depicted in the synopsis.

savePrefs is undoable, NOT queryable, and NOT editable.

This command saves preferences to disk. If no flags are specified then all pref types get saved out.

Return value

None

Flags

colors, general, hotkeys, menuSets, plugins, uiLayout
Long name (short name) Argument types Properties
colors(c) boolean
Save the color prefs to disk
general(g) boolean
Save the general prefs to disk (optionVars)
hotkeys(hk) boolean
Save the hotkeys to disk
menuSets(ms) boolean
Save the menuSet preferences to disk
plugins(pl) boolean
Save the plug-in prefs to disk
uiLayout(ui) boolean
Save each window's size and position to disk
Flag can appear in Create mode of command Flag can appear in Edit mode of command
Flag can appear in Query mode of command Flag can have multiple arguments, passed either as a tuple or a list.

Python examples

import maya.cmds as cmds

# only save the hotkeys
cmds.savePrefs( hotkeys=True )

# save everything
cmds.savePrefs()

# only save ui layout prefs
cmds.savePrefs( uiLayout=True )
