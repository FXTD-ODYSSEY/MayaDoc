command (Python) MEL version

greasePencilCtx

In categories: General, Contexts

Go to: Synopsis. Return value. Flags. Python examples.

Synopsis

greasePencilCtx([autoCreateFrames=boolean], [canDraw=boolean], [createOrEditFrame=int], [exists=boolean], [exportArchive=[string, string]], [fileTextureSize=int], [greasePencilType=int], [image1=string], [image2=string], [image3=string], [importArchive=string], [makeStroke=uint], [removeFrame=int], [resetBrushes=boolean], [rgbcolor=[float, float, float]], [sequenceNodeName=string])

Note: Strings representing object names and arguments must be separated by commas. This is not depicted in the synopsis.

greasePencilCtx is undoable, queryable, and editable.

This is a tool context command for the grease pencil tool.

Return value

None

In query mode, return type is based on queried flag.

Flags

autoCreateFrames, canDraw, createOrEditFrame, exists, exportArchive, fileTextureSize, greasePencilType, image1, image2, image3, importArchive, makeStroke, removeFrame, resetBrushes, rgbcolor, sequenceNodeName
