command (MEL) Only available in MEL

dot

In categories: Language, Math

Go to: Synopsis. Return value. Related. MEL examples.

Synopsis

dot vector vector

dot is undoable, NOT queryable, and NOT editable.

This command returns the dot product of two vectors.

Return value

floatDot product value

Related

angle, cross, mag

MEL examples

vector $u=<<1,2,-2>>, $v=<<3,0,1>>;
$x=dot($u, $v);
// Result: 1 //
