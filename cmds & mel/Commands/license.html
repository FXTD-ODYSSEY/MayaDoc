command (MEL) Python version

license

In categories: General

Go to: Synopsis. Return value. Flags. MEL examples.

Synopsis

license [-borrow] [-info] [-isBorrowed] [-isExported] [-isTrial] [-licenseMethod] [-productChoice] [-return] [-showBorrowInfo] [-showProductInfoDialog] [-status] [-usage]

license is undoable, NOT queryable, and NOT editable.

This command displays version information about the application if it is executed without flags. If one of the above flags is specified then the specified version information is returned.

Return value

stringThe application's license information.

Flags

borrow, info, isBorrowed, isExported, isTrial, licenseMethod, productChoice, return, showBorrowInfo, showProductInfoDialog, status, usage
