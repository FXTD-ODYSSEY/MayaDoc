Synopsis

mag vector

mag is NOT undoable, NOT queryable, and NOT editable.

This command returns the magnitude (or length) of its argument.

Return value

floatMagnitude value

Related

angle, cross, dot

MEL examples

mag <<7, 8, 9>>;
// Result: 13.9284 //
