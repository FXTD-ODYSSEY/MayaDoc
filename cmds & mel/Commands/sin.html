command (MEL) Only available in MEL

sin

In categories: Language, Math

Go to: Synopsis. Return value. Related. MEL examples.

Synopsis

sin float

sin is NOT undoable, NOT queryable, and NOT editable.

Returns the sine of the given value.

Also available:
asin : Returns the arc sine in the range -PI/2 to PI/2.
sinh : Hyperbolic function.
asinh : Inverse hyperbolic function.
sind : Returns value in degrees.
asind : Returns value in degrees.

Return value

floatSinus value

Related

cos, tan

MEL examples

sin (deg_to_rad(90));
