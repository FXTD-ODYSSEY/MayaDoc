Synopsis

string shortNameOf(string $object)

This procedure returns the short name of $object. This name may not be unique.

Return value

None

MEL examples

		//	Create a sphere.

		//

		string $spheres[] = `polySphere -name "sphere1"`;
		string $sphere = $spheres[0];

		shortNameOf($sphere);
		//	Result: sphere1 //
