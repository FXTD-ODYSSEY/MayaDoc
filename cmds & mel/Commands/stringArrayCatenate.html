Synopsis

string[] stringArrayCatenate( string $string1[], string $string2[] )

Returns a new string array which is the result of concatenating the two given string arrays.

Return value

None

Arguments
Variable Name Variable Type Description
$string1string[]First string array
$string2string[]Second string array to be added to the first.

MEL examples

	string $string1[] = {"light1", "light2"};
	string $string2[] = {"light3","light4","light5"};
	stringArrayCatenate($string1, $string2);
	// Result: light1 light2 light3 light4 light5 //
