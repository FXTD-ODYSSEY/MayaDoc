randomizeFollicles

In categories: Effects, Hair

Go to: Synopsis. Return value. MEL examples.

Synopsis

randomizeFollicles range

This mel procedure randomizes the positions of selected follicles in uv space. The passed in range value controls the scale of the randomization applied in normalized uv space.

Return value

None

Arguments
Variable Name Variable Type Description
None.
