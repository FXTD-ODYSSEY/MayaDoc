Synopsis

reparentStrokes

This transfers all strokes on the first selected object to the second selected object. There should be exactly two objects selected and both objects should be the same type: either nurbsSurface or mesh. One should first select the target and then the destination object, but if only one object has strokes on it then the order does not matter. The toplogy need not match between the objects, but the uvs should match if one wishes the strokes to transfer to matching positions.

Return value

None
