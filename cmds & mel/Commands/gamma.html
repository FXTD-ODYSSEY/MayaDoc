command (MEL) Only available in MEL

gamma

In categories: Language, Math

Go to: Synopsis. Return value. MEL examples.

Synopsis

gamma float

gamma is NOT undoable, NOT queryable, and NOT editable.

Let x be the floating point intput argument, then gamma returns ln|(GAMMA(x))|, where GAMMA is the mathematical gamma function.

This command is implemented using the built-in system function.

Return value

floatGamma value

MEL examples

gamma 2.5;
// Result:, 0.284683 //
