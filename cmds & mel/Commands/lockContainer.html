Synopsis

lockContainer

This script locks the unpublished attributes and nodes in the container associated with the selected nodes, as well as locking the container itself.

Return value

None

Arguments
Variable Name Variable Type Description
$valintif 1, lock the unpublished attributes and nodes in the container and the container itself, if 0 unlock

MEL examples

	// lock the container that locator1 is in

	select -r locator1;
	lockContainer 1;
