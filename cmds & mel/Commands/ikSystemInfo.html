ikSystemInfo

In categories: Animation, IK

Go to: Synopsis. Return value. Related. Flags. MEL examples.

Synopsis

ikSystemInfo [-globalSnapHandle] boolean

ikSystemInfo is undoable, queryable, and NOT editable.

This action modifies and queries the current ikSystem controls.

Return value

None

In query mode, return type is based on queried flag.

Related

connectJoint, ikHandle, ikHandleDisplayScale, ikSolver, ikSystem, insertJoint, joint, jointCluster, jointDisplayScale, jointLattice, mirrorJoint, removeJoint

Flags

globalSnapHandle
Long name (short name) Argument types Properties
-globalSnapHandle(-gsh)
If this flag is off, all ikHandles will not be snapped.
Flag can appear in Create mode of command Flag can appear in Edit mode of command
Flag can appear in Query mode of command Flag can be used more than once in a command.

MEL examples

//Enable global handle snapping
//
ikSystemInfo -gsh on;
