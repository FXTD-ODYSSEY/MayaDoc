script (MEL) Only available in MEL

selectCurveCV

In categories: Modeling, Curves

Go to: Synopsis. Return value. MEL examples.

Synopsis

selectCurveCV ( string $CVsToSelect )

Selects the specified CV on active curve.

Return value

None

Arguments
Variable Name Variable Type Description
$CVsToSelect: valid strings are "first", "last" and "all".

MEL examples

 // Select the first cv.

 selectCurveCV("first");

 // Select the last cv.

 selectCurveCV("last");

 // Select all cv's.

 selectCurveCV("all");
