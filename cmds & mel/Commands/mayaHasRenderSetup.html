mayaHasRenderSetup

In categories: Rendering

Go to: Synopsis. Return value. Keywords. Flags. MEL examples.

Synopsis

mayaHasRenderSetup [-enableCurrentSession boolean] [-enableDuringTests boolean]

mayaHasRenderSetup is NOT undoable, queryable, and editable.

This command controls and queries render setup states.

Return value

None

In query mode, return type is based on queried flag.

Keywords

render, setup

Flags

enableCurrentSession, enableDuringTests
Long name (short name) Argument types Properties
-enableCurrentSession(-ecs) boolean
Enables or disables render setup for this Maya session only. This flag should only be called during Maya intialization. This flag is for internal use only, may change at any time and is unsupported.
-enableDuringTests(-edt) boolean
Switches render setup for this Maya session only, as legacy render layer mode is assumed during testing. This flag is for internal use only, may change at any time and is unsupported.
Flag can appear in Create mode of command Flag can appear in Edit mode of command
Flag can appear in Query mode of command Flag can be used more than once in a command.

MEL examples


//Returns whether render setup mode is enabled or disabled
mayaHasRenderSetup

//Query the state renderSetupEnableCurrentSession
mayaHasRenderSetup -query -enableCurrentSession

//Query the state renderSetupEnableDuringTests
mayaHasRenderSetup -query -enableDuringTests

//Enable renderSetupEnableCurrentSession
mayaHasRenderSetup -edit -enableCurrentSession true

//Disable renderSetupEnableCurrentSession
mayaHasRenderSetup -edit -enableCurrentSession false

//Enable renderSetupEnableDuringTests
mayaHasRenderSetup -edit -enableDuringTests true

//Disable renderSetupEnableDuringTests
mayaHasRenderSetup -edit -enableDuringTests false
