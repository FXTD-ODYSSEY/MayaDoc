command (MEL) Only available in MEL

abs

In categories: Language, Math

Go to: Synopsis. Return value. MEL examples.

Synopsis

abs int|float|vector

abs is undoable, NOT queryable, and NOT editable.

Returns the absolute value of its argument. In the case of a vector argument, it returns a vector composed of the absolute value of of its components.

Return value

intWhen the argument is an integer
floatWhen the argument is a float
vectorWhen the argument is a vector

MEL examples

$f=-123;
abs $f;
// Result: 123 //

abs <<-1, -2.432, 555>>;
// Result: <<1, 2.432, 555>>  //
