Synopsis

makeTubeOn ()

This function turns tubes on for the current template brush, and sets the tube generation parameters such that there is little change in appearance. A single tube is created that exactly follows the brush path and has the same width as the brush.

Return value

None

Arguments
Variable Name Variable Type Description
None.
