command (MEL) Only available in MEL

angle

In categories: Language, Math

Go to: Synopsis. Return value. Related. MEL examples.

Synopsis

angle vector vector

angle is undoable, NOT queryable, and NOT editable.

This command returns the angle (in radians) between two vectors.

Return value

floatMeasured angle in radian

Related

cross, dot, mag

MEL examples

$u=<<2,-1, 1>>; $v=<<1,1,2>>;
angle $u $v;
// Result: 1.0472 //
rad_to_deg(`angle $u $v`);
// Result: 60 //
